# 战斗系统

## 概述

战斗不是本游戏的核心玩法，而是**人生路线中的一种遭遇**。战斗以策略准备为主，执行过程自动进行。

## 设计原则

- **战前策略 > 战斗操作**：带什么装备、学什么武功、吃什么丹药，比战斗中的操作更重要
- **战斗是选择，不是强制**：大部分遭遇可以选择回避、谈判、逃跑
- **轻量呈现**：MVP阶段不做战斗场景，文字结果即可

## 战斗公式

### 伤害计算

```
基础伤害 = 攻击力 × 武功倍率
防御减免 = 防御力 × 护甲系数
最终伤害 = (基础伤害 - 防御减免) × 随机浮动(0.9~1.1)
```

### 暴击

```
暴击概率 = 武功暴击率 + 装备暴击加成
暴击伤害 = 最终伤害 × 1.5
```

### 闪避

```
闪避概率 = (轻功差值 / 100) + 装备闪避加成
闪避成功：伤害为0
```

### 逃跑

```
逃跑概率 = (己方轻功 - 敌方轻功) / 100 + 基础概率(0.3)
```

## 战斗结果

| 结果 | 效果 |
|------|------|
| 胜利 | 获得战利品、经验值 |
| 失败 | 损失部分金钱、回到客栈恢复 |
| 逃跑 | 无奖励无惩罚 |
| 谈判成功 | 根据谈判结果（给钱/化敌为友/获取信息） |
