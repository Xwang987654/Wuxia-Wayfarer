## 历史

```
百年前：
  清风镇只是一个路边茶棚，供过路商旅歇脚。

六十年前：
  有赵姓铁匠在此落脚，开了第一间铺子。
  因为位置好（南北要道），渐渐有人聚居。
  形成了一个小村落。

四十年前：
  村落发展为镇，有了集市。
  赵家铁匠的儿子开始教年轻人练拳（铁掌门前身）。
  官府设了一个衙门，派了一个捕头。

二十年前：
  清风镇成为方圆百里最大的集镇。
  商旅往来频繁，铺子越开越多。
  铁掌门正式成为门派。
  镇上有了客栈、药铺、茶馆。

如今：
  清风镇是江南到中原的必经之地。
  南来北往的商旅在此歇脚、交易、打听消息。
  铁掌门是镇上最大的武力组织。
  衙门维持基本治安。
  各色人等汇聚于此，信息交汇，机会与危险并存。
```

### 地点描述

#### 集市中心

镇中心的广场，每天上午和傍晚最热闹。

```
广场不大，青石板铺地，中央有一棵老槐树。
老槐树下挂着一块木板——悬赏板，上面贴着各种告示。
广场四周是摊位，卖什么的都有：蔬菜、布料、杂货、小吃。
南来北往的商人在这里摆摊，吆喝声此起彼伏。
偶尔有卖艺的、算命的、唱曲的在此讨生活。
```

#### 铁匠铺

张大锤的铺子，清风镇唯一的铁匠铺。

```
一间烟熏火燎的铺面，门口挂着各种铁器。
炉火终日不熄，叮叮当当的打铁声从早响到晚。
铺子里摆满了刀剑锄犁，墙上挂着几把好剑——那是张大锤的得意之作。
后院是锻造间，有砧板、风炉、淬火用的水缸。
```

#### 药铺

李药师的药铺，镇上唯一的医药之所。

```
门面不大，但一进门就是扑鼻的药香。
墙上密密麻麻的小抽屉，每个里面装着不同的药材。
柜台后面有一排药罐，里面泡着各种药酒。
后院有一间小屋，是李药师炼丹的地方，平时不让人进。
```

#### 客栈

王掌柜的客栈，清风镇唯一的住宿之所。

```
两层小楼，一楼是大堂，二楼是客房。
大堂里摆着七八张桌子，白天供食客吃饭，晚上供住客休息。
门口挂着一面酒旗，写着"清风客栈"四个字。
柜台后面有一面墙，上面贴满了各种告示和留言——这也是消息最灵通的地方。
夜晚，大堂里灯火通明，过路商旅在此喝酒聊天，交换各地消息。
```

#### 铁掌门

清风镇最大的武馆，也是唯一的门派。

```
位于镇子东北角，占地不小。
大门上挂着一块匾，写着"铁掌门"三个字，笔力遒劲。
门前有两个石狮子，是赵铁山年轻时从山里搬来的。
进去是一个大院子，地面是夯实的泥土，是弟子们练功的地方。
院子四周摆着木桩、石锁、沙袋等练功器械。
正厅是议事厅，平时赵铁山在这里教弟子。
后院是弟子们的住处和赵铁山的书房。
```

#### 衙门

清风镇的官府所在地。

```
位于镇子西北角，和铁掌门遥遥相对。
门前有一面鸣冤鼓，落满灰尘——已经很久没人敲过了。
衙门不大，就一个院子：前堂办公，后堂住人。
陈捕头手下有四个捕快，管着整个清风镇的治安。
衙门后面有一个小牢房，关过最严重的犯人也不过是醉酒闹事的。
```

## 镇上的一天

```
清晨（天刚亮）：
  铁掌门弟子在院子里练功，喊声传出很远。
  李药师背着药篓去后山采药。
  老农挑着蔬菜去集市占位置。

上午（日上三竿）：
  集市渐渐热闹起来。
  铁匠铺叮叮当当开始打铁。
  商旅从官道上过来，进镇歇脚。
  衙门开门办公（其实没什么事）。

午后（日头最毒）：
  集市人少了，商贩在摊位旁打盹。
  客栈大堂里几个旅人喝酒聊天。
  铁掌门弟子午休后继续练功。
  李药师在药铺后院炼丹。

傍晚（夕阳西下）：
  集市又热闹起来——这是第二波高峰。
  过路的商旅这时候进镇，找客栈住下。
  铁掌门弟子收功，有人去客栈喝酒。
  衙门关门，捕快回家。

夜晚（月上柳梢）：
  客栈大堂灯火通明，是消息交汇的时刻。
  镇上人家关门闭户。
  偶尔有醉汉从客栈出来，摇摇晃晃回家。
  后山方向偶尔传来几声狼嚎。
```

## 镇上的规矩

```
明面上：
  • 衙门管治安，铁掌门管武林事
  • 做生意要交税（不多）
  • 不许在镇上动刀子
  • 有纠纷找衙门或铁掌门调解

暗地里：
  • 铁掌门说了算——赵铁山的话比捕头管用
  • 青竹帮偶尔来镇上，只要不惹事就没人管
  • 黑市在夜晚的客栈后院（卖些来路不明的东西）
  • 有些铺子表面做生意，暗地里做别的（后续可展开）
```

## 信息从哪里来

清风镇是枢纽，信息从四面八方汇聚：

```
┌──────────────┬──────────────────────────────────────────────┐
│  来源        │  信息类型                                    │
├──────────────┼──────────────────────────────────────────────┤
│  过路商旅    │  各地物价、商贸消息、路上见闻                │
│  江湖人      │  武林动态、门派纷争、高手行踪                │
│  衙门公文    │  官府消息、通缉令、政策变化                  │
│  乞丐/流浪者 │  底层消息、小道传闻                          │
│  客栈闲聊    │  酒后真言、各种真假难辨的消息                │
└──────────────┴──────────────────────────────────────────────┘

玩家在清风镇可以听到各种消息，这些消息指向不同的方向：
  • "听说临河镇最近出了件大事" → 未来的城镇
  • "铁掌门的赵掌门年轻时可不简单" → NPC故事
  • "官府最近在招人" → 衙门路线
  • "后山那帮人最近不太安分" → 青竹帮任务
```
