# 任务系统

## 概述

任务不是线性主线，而是**散布在世界中的机会**。玩家可以选择接取、忽略或延迟完成，没有强制性。

## 任务类型

### 悬赏任务

- **来源**：集市中心悬赏板
- **特点**：公开任务，任何人可接
- **示例**：
  - "清剿后山山贼，赏银50两"
  - "寻找失踪的采药人"
  - "护送商队到临河镇"
- **刷新**：每日刷新部分任务

### NPC 支线任务

- **来源**：与NPC对话触发
- **特点**：需要好感度或特定条件
- **示例**：
  - 铁匠："帮我找一块玄铁，我给你打把好剑"
  - 药师："后山有种稀有草药，帮我采来"
  - 客栈掌柜："最近总有人赊账，帮我查查"
- **解锁条件**：好感度达到友好

### 官府任务（衙门路线专属）

- **来源**：衙门捕头
- **特点**：查案、断案类，有剧情
- **示例**：
  - "镇东头发生了盗窃案，去调查一下"
  - "有人举报赌场，去查封"
  - "通缉犯出现在附近，去抓捕"
- **成长性**：从小案到大案，逐步晋升

### 门派任务（门派路线专属）

- **来源**：门派师傅
- **特点**：修炼、比武类
- **示例**：
  - "去后山闭关修炼三天"
  - "与师兄切磋武艺"
  - "代表门派参加镇上比武"
- **成长性**：从杂役到亲传弟子

### 随机事件

- **来源**：探索、时间触发
- **特点**：不可预期，有惊喜感
- **示例**：
  - 路遇受伤的江湖人，选择救助或无视
  - 发现一处隐秘山洞
  - 遇到江湖骗子，选择揭穿或配合
  - 集市上出现神秘商人卖稀有物品

## 任务结构

```yaml
任务ID: quest_001
名称: 清剿山贼
类型: 悬赏
发布人: 悬赏板
描述: 后山最近聚集了一伙山贼，骚扰来往商旅。请前往清剿。

目标:
  - 击败山贼 x3

条件:
  无（任何人都可接取）

奖励:
  铜钱: 500
  声望: 侠义 +10

失败条件:
  玩家生命归零（可重新尝试）

分支:
  - 谈判成功：获得山贼藏宝图
  - 逃跑：任务仍可再次接取
```

## 任务流程

```
发现任务
   │
   ▼
┌──────────────┐
│  接取任务     │  ← 可选择接或不接
└──────┬───────┘
       │
       ▼
┌──────────────┐
│  执行任务     │  ← 可随时放弃
└──────┬───────┘
       │
       ▼
┌──────────────┐
│  完成/失败    │  ← 获得奖励或重新尝试
└──────┬───────┘
       │
       ▼
  任务记录存档
```
