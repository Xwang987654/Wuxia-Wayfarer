## 内力

### 什么是内力

内力，又称真气、内劲。是人通过修炼，在体内凝聚的一股力量。

常人体内也有真气，但微弱到可以忽略。武者通过吐纳、导引、打坐等方法，日积月累，将真气凝聚壮大。

### 内力的作用

```
┌──────────────┬──────────────────────────────────────────────┐
│  作用        │  说明                                        │
├──────────────┼──────────────────────────────────────────────┤
│  增强攻击    │  将真气灌注于拳掌兵器，威力倍增              │

└──────────────┴──────────────────────────────────────────────┘
```

### 内力修炼

内力修炼是一个漫长的过程，没有捷径（表面上看）：

```
入门：能感觉到体内有气在流动
  ↓ （数月）
初成：能将真气灌注于四肢，增强力量
  ↓ （数年）
小成：真气可护体，普通刀剑难伤
  ↓ （十数年）
大成：真气浑厚，隔空可伤人
  ↓ （数十年）
宗师：真气与天地相通，返璞归真
```

大多数人穷其一生，也不过"初成"到"小成"之间。能到"大成"者，已是江湖一流高手。"宗师"境界，百年难出一个。

### 门派差异

不同门派的内功心法侧重不同：

```
铁掌门：刚猛内力，走阳刚路线
  → 真气浑厚，爆发力强，但持久性一般
  → 适合近身搏击

（其他门派各有特色，后续扩展）
```

## 招式

### 什么是招式

招式，即攻防的具体技法。一拳怎么打、一剑怎么刺、一步怎么退，都是招式。

招式看似简单，实则蕴含前人无数次实战的经验。一个好的招式，能在最小的消耗下造成最大的伤害。

### 招式与内力的关系

```
内力强 + 招式弱 = 蛮力（有力使不出）
内力弱 + 招式强 = 巧劲（四两拨千斤，但上限低）
内力强 + 招式强 = 上乘武学

类比：
  内力 = 发动机排量
  招式 = 驾驶技术
  排量大但技术差 = 浪费动力
  技术好但排量小 = 够用但跑不快
  排量大且技术好 = 又快又稳
```

## 两套武学体系

天下武学分为两大体系：

### 门派武学

```
特点：
  • 师徒传承，有完整的教学体系
  • 内功心法 + 招式套路 + 实战训练
  • 有门派归属感和规矩
  • 修炼速度适中
  • 上限高（大门派有镇派绝学）

获取方式：
  • 拜入师门
  • 通过门派考核
  • 逐步解锁更高深的武功
```

### 官府武学

```
特点：
  • 标准化教学，类似军中训练
  • 强调实战效率，不追求精妙
  • 制式化，所有官府武者学的都差不多
  • 修炼速度较快（有成熟的训练方法）
  • 上限较低（缺乏高深的内功心法）

获取方式：
  • 入职衙门/官府机构
  • 通过官府考核
  • 按等级逐步解锁

  官府武学的优势：
  ✅ 入门快，3个月可形成战斗力
  ✅ 有统一标准，不会出现"学歪了"的情况
  ✅ 有官府背景，行事方便

  官府武学的劣势：
  ❌ 上限低，遇到真正的高手差距明显
  ❌ 缺乏变化，容易被摸透套路
  ❌ 没有独门绝技，千人一面
```

### 二者对比

```
                门派武学              官府武学
  ─────────────────────────────────────────────
  入门难度      较高（需拜师）        较低（入职即可）
  修炼速度      中等                  较快
  上限          高                    中等
  独特性        强（各有绝学）        弱（标准化）
  自由度        高（可自选路线）      低（按部就班）
  组织归属      门派                  官府

  在清风镇（MVP）：
  二者实力均衡，各有优劣
  玩家选门派还是官府，是风格偏好而非强弱之分
```

## 江湖规矩

### 以武会友

门派之间、高手之间，可以"切磋"代替"拼命"。切磋有规矩：点到为止，不伤性命。但总有人"失手"。

### 江湖恩怨

江湖人解决恩怨的方式：
- **讲理**：先礼后兵
- **切磋**：比武定胜负
- **决斗**：生死不论
- **调解**：找德高望重的前辈调停

### 门派与官府

表面上：
- 门派不干涉官府办案
- 官府不干涉门派收徒
- 各管各的，井水不犯河水

暗地里：
- 官府武学总署在渗透小门派
- 大门派在暗中抵制
- 一些小门派已经被官府"扶持"，实际上是傀儡
