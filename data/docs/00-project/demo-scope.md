# Demo Scope

本文件是第一版 demo 的唯一范围定义。任何文档、数据和实现如果与本文件冲突，以本文件为准。

## 必须做到

- 玩家可以进入清风镇。
- 玩家可以在至少 3 个地点间行动。
- 玩家可以与 3 个 NPC 对话。
- 玩家可以接取并完成 1 个任务。
- 玩家可以获得或失去物品、铜钱。
- 玩家可以看到背包或状态中的变化。
- 游戏时间至少能从一个时段推进到下一个时段。
- 所有可玩内容都来自 `data/game-runtime/`。

## 不做内容

- 门派系统。
- 战斗系统。
- 声望系统。
- 复杂经济和开店。
- 隐藏武功。
- 大地图旅行。
- 深度季节天气模拟。
- 多结局或多路线人生。
- 随机事件池。
- 技能、装备、制造、采矿、黑市。

## Demo 内容

- 地图：清风镇。
- 地点：镇口、青石街、林家药铺、清风客栈、溪边草坡。
- NPC：药师林、周掌柜、赵守卫。
- 任务：帮药师林采回 3 份草药。
- 物品：铜钱、草药、馒头。
- 时间：清晨、午间、傍晚、夜晚。

## 完成判断

玩家从初始状态出发，可以不看外部说明完成一次采草药任务，并看到奖励和时间变化，即视为第一版 demo 的玩法闭环成立。
