# 经济系统

## 概述

经济系统是经商路线的核心，也是所有路线的基础货币流通系统。玩家通过交易、开店、制造等方式获取财富。

## 货币

| 货币 | 说明 | 获取方式 |
|------|------|---------|
| 铜钱 | 基础货币 | 任务奖励、战斗掉落、交易利润 |
| 银两 | 高级货币（1银两=1000铜钱） | 大宗交易、高难度任务 |

## 交易系统

### 集市买卖

- 集市中心有多个摊位NPC，各有收购和出售物品
- 物品价格**每日浮动**（基于供需模拟）
- 同一物品在不同NPC处价格不同（信息差=赚钱机会）

### 价格浮动机制

```
今日价格 = 基础价格 × 供需系数 × 季节系数 × 天气系数 × 随机浮动

供需系数：买的人多则涨，卖的人多则跌
季节系数：应季物品便宜，反季物品贵
天气系数：雨天雨伞贵，冬天棉衣贵
随机浮动：±10%
```

### 跑商

- 不同地点（城镇）之间有价差
- 玩家可以低买高卖赚取差价
- 跑商有风险：路上可能遭遇劫匪
- 跑商利润与口才属性相关（议价能力）

## 开铺子

### 流程

```
① 选择地点（租/买铺面）
   │
   ▼
② 选择经营类型
   │  • 药铺（卖丹药/药材）
   │  • 铁匠铺（卖武器/防具）
   │  • 茶馆（卖茶/食物，打听消息）
   │  • 杂货铺（什么都卖）
   │
   ▼
③ 进货
   │  • 从NPC批发采购
   │  • 自己制造
   │  • 从探索中获取
   │
   ▼
④ 定价
   │  • 定价高→利润高但客流少
   │  • 定价低→客流多但利润薄
   │
   ▼
⑤ 运营
      • NPC自动来买（基于铺子声望+价格+位置）
      • 每日结算收入
      • 铺子声望随时间积累
```

### 铺子属性

| 属性 | 说明 |
|------|------|
| 声望 | 影响客流量，通过好货/好价/时间积累 |
| 库存 | 货架上的商品，卖完需补货 |
| 位置 | 影响基础客流 |
| 规模 | 影响可陈列商品数量 |

## 制造系统（轻量版）

### 制造流程

```
原料 → 加工 → 成品

草药 ──→ 炼丹 ──→ 丹药
矿石 ──→ 打铁 ──→ 武器/防具
布料 ──→ 裁缝 ──→ 衣服
茶叶 ──→ 炒制 ──→ 茶叶（高级）
```

### 制造规则

- 需要对应工具（丹炉/铁砧/缝纫台）
- 需要配方（从NPC购买/任务获得/探索发现）
- 成品品质受手艺属性影响
- 制造消耗体力
- 失败率存在（可通过提升手艺降低）

## 经济平衡

### 收入来源

| 来源 | 金额范围 | 频率 |
|------|---------|------|
| 战斗掉落 | 少量 | 每次战斗 |
| 任务奖励 | 中等 | 每个任务 |
| 交易利润 | 看操作 | 随时 |
| 铺子收入 | 稳定 | 每日 |
| 制造利润 | 中等 | 每次制造 |

### 支出项目

| 项目 | 金额范围 | 频率 |
|------|---------|------|
| 购买装备 | 中等 | 按需 |
| 购买丹药 | 少量 | 频繁 |
| 铺子租金 | 固定 | 每日/每月 |
| 拜师学费 | 一次性 | 按需 |
| 住宿 | 少量 | 每日 |
| 跑商进货 | 大量 | 按需 |

### 目标经济曲线

```
财富
  │
  │                                    ╱ 富甲一方
  │                              ╱────╱
  │                        ╱────╱
  │                  ╱────╱
  │            ╱────╱
  │      ╱────╱
  │ ╱───╱
  │╱
  └──────────────────────────────────── 时间
  穷困    温饱    小康    富裕    豪富
```

初期赚钱慢，中期有稳定收入来源后加速，后期财富积累快但有大额支出目标（买房、开分店等）。
