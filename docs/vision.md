# 愿景与目标

## 一句话愿景

> 在像素化的江湖中，过自己想过的武侠人生。

## 核心体验目标

玩家打开这个游戏，不是为了通关、不是为了刷装备、不是为了操作战斗——而是为了**在武侠世界里过一种自己选择的生活**。

可以是：
- 一个清晨在竹林练剑、午后在茶馆听书的散人
- 一个从学徒做到掌门的门派弟子
- 一个白日查案、夜晚在自家药铺算账的捕快
- 一个走南闯北、低买高卖的行商

**没有唯一正确的玩法，只有你选择的人生。**

## 设计原则

### 1. 自由优先
玩家可以随时切换人生方向，不被锁定在某一条路线上。门派弟子可以叛出门派，捕快可以辞职经商，散人可以随时拜师。

### 2. 养生不肝
- 没有体力上限逼你每天上线
- 没有排行榜制造焦虑
- 时间自然流逝，错过了等明年再来
- 挂机也有收益（铺子自动卖货、弟子自动练功）

### 3. 世界活着
NPC有自己的作息、喜好、关系网。你不在的时候世界也在运转——商队来往、季节更替、江湖传闻流传。

### 4. 选择有重量
每一个选择都有后果，但不是惩罚性的"坏结局"。加入衙门意味着不能随意作恶，但获得了官府资源。选择当散人意味着自由，但没有组织庇护。

### 5. 简单但有深度
操作简单（点击/选择为主），但系统之间有联动——天气影响物价、季节影响采药、声望影响NPC态度、武功影响战斗结果。

## 目标玩家

- 喜欢武侠/仙侠题材的玩家
- 喜欢模拟经营、养成类游戏的玩家
- 不喜欢高强度操作、偏好轻松体验的玩家
- 喜欢高自由度、不被主线束缚的玩家
- 烟雨江湖玩家中觉得自由度不够的那部分人

## 项目目标

### 阶段一：MVP（清风镇）
验证核心玩法循环：时间流逝 + 基础交互 + 一条完整的人生路线体验。

### 阶段二：扩展（江南地区）
多城镇、跑商系统、更多门派和NPC，验证经济系统的可玩性。

### 阶段三：完整江湖
多个地区、完整世界观、复杂势力关系、多结局。

## 非目标（明确不做）

- 不做硬核动作战斗
- 不做PVP
- 不做社交/联机（至少现阶段不做）
- 不做3D
- 不做付费抽卡/氪金系统
- 不做复杂到需要教程才能玩的系统
