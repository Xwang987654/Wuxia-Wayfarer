# MVP 里程碑

## 目标

完成清风镇的核心玩法循环，验证游戏的可玩性。

## MVP 范围

**一句话**：玩家可以在清风镇中自由行动、与NPC交互、完成简单任务、体验一天的时间流逝。

### 包含

| 模块 | 功能 | 优先级 |
|------|------|--------|
| 地图 | 清风镇内所有地点 + 镇外2-3个探索点 | P0 |
| 角色 | 创建角色、基础属性、装备系统 | P0 |
| 时间 | 日夜循环 + 四季 + 天气 | P0 |
| NPC | 5-8个功能性NPC + 基础对话 + 交易 | P0 |
| 交易 | 集市买卖、价格浮动 | P0 |
| 战斗 | 文字结果输出（自动战斗） | P0 |
| 门派 | 铁掌门：拜师、学2-3个拳法 | P1 |
| 衙门 | 入职、接2-3个官府任务 | P1 |
| 探索 | 镇外2-3个探索点 + 3-5个随机事件 | P1 |
| 任务 | 悬赏板3-5个任务 | P1 |
| 经营 | 简化版开店（药铺或铁匠） | P2 |
| 声望 | 侠义/恶名基础系统 | P2 |

### 不包含（后续版本）

- 多城镇（临河镇、落霞城）
- 跑商系统
- 复杂制造系统
- 复杂任务链
- 门派完整成长线
- 隐藏武功
- 季节专属大型事件

## 功能拆解

### P0：核心（必须完成）

#### 地图系统
- [ ] 清风镇地图渲染（2.5D等距）
- [ ] 地点切换（进入/离开建筑）
- [ ] 基础碰撞和路径

#### 角色系统
- [ ] 角色创建（选出身）
- [ ] 基础属性（生命/体力/内力/攻击/防御/轻功）
- [ ] 装备系统（武器/防具）
- [ ] 物品背包

#### 时间系统
- [ ] 时间流逝（10分钟/天）
- [ ] 日夜5时段
- [ ] 四季循环（6天/季）
- [ ] 基础天气（晴/雨/雪）
- [ ] 时间快进功能

#### NPC系统
- [ ] 5-8个NPC（张大锤、李药师、王掌柜、赵掌门、陈捕头、刘商人、老乞丐）
- [ ] NPC日程（按时段移动/行为变化）
- [ ] 基础对话（选择式）
- [ ] 好感度系统

#### 交易系统
- [ ] 集市买卖界面
- [ ] 价格每日浮动
- [ ] NPC差异化库存

#### 战斗系统
- [ ] 战斗触发（探索中遭遇）
- [ ] 自动战斗计算
- [ ] 战斗结果UI（文字描述）
- [ ] 战利品获取

#### 存档系统
- [ ] 游戏存档
- [ ] 游戏读档

### P1：重要（应该完成）

#### 门派路线
- [ ] 铁掌门拜师流程
- [ ] 学习2-3个拳法
- [ ] 门派任务2-3个

#### 衙门路线
- [ ] 衙门入职流程
- [ ] 官府任务2-3个

#### 探索系统
- [ ] 镇外2-3个探索区域
- [ ] 随机事件3-5个
- [ ] 采药/采集功能

#### 任务系统
- [ ] 悬赏板
- [ ] 3-5个悬赏任务
- [ ] 任务日志UI

### P2：锦上添花（可以完成）

#### 经营系统
- [ ] 租铺子
- [ ] 简化版开店（进货→定价→自动售卖）

#### 声望系统
- [ ] 侠义/恶名量表
- [ ] 声望对NPC的影响

## 技术决策点

| 决策项 | 选项 | 截止时间 |
|--------|------|---------|
| 技术栈 | Web/Godot/Phaser | 开发前 |
| 渲染方案 | Canvas/WebGL/引擎内置 | 开发前 |
| 数据格式 | JSON/YAML | 开发前 |
| 存档方案 | LocalStorage/文件 | 开发前 |

## 质量标准

- 所有P0功能可正常运行
- 无崩溃性bug
- 存档读档正常
- 时间系统准确运行
- NPC日程按预期变化
- 交易价格浮动正常
