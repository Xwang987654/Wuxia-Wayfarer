# 世界观与地图

## 世界观概述

故事发生在一个架空的中国古代江湖世界。天下太平已久，但江湖暗流涌动——各大门派明争暗斗，官府试图收拢江湖势力，商路繁荣但匪患未绝。

玩家是一个初入江湖的年轻人，身世平凡，来到江南的清风镇，开始自己的人生。

## 世界地图（可扩展架构）

```
Phase 1：清风镇
Phase 2：江南地区（清风镇 + 临河镇 + 落霞城）
Phase 3：更大世界（中原 + 塞外 + 南疆 + 江南）
```

### 地点详细说明

#### 镇内地点

| 地点 | 描述 | 功能 | 对应路线 |
|------|------|------|---------|
| 集市中心 | 镇中心广场，人流密集 | 悬赏板、交易、NPC聚集 | 通用 |
| 主街道 | 连接各建筑的主要道路 | 移动通道、随机NPC | 通用 |
| 铁掌门 | 清风镇最大的武馆 | 拜师、学拳法、门派任务 | 门派 |
| 药铺 | 李药师经营的药铺 | 买卖药材丹药、炼丹 | 经商 |
| 铁匠铺 | 张大锤的打铁铺 | 买卖武器装备、锻造 | 经商 |
| 客栈 | 清风镇唯一的客栈 | 住宿、休息、打听消息 | 散人 |
| 衙门 | 清风镇官府 | 入职、接官府任务 | 衙门 |

#### 镇外地点

| 地点 | 描述 | 功能 | 可探索内容 |
|------|------|------|---------|
| 竹林小径 | 离开小镇的第一条路 | 探索、采药 | 随机事件、草药采集 |
| 溪边药田 | 溪流旁的野生药田 | 采药 | 稀有草药、遇到药师学徒 |
| 后山密林 | 山贼出没的密林 | 战斗、探索 | 山贼窝点、隐藏宝箱 |
| 古庙废墟 | 荒废的古庙 | 探索 | 隐藏武功秘籍、神秘NPC |
| 官道 | 连接外界的大路 | 跑商、旅行 | 商队、劫匪、旅人 |

### 天气与地点

| 天气 | 地点影响 |
|------|---------|
| 大雪 | 后山密林不可进入 |
| 暴雨 | 溪边药田不可进入 |
| 雾气 | 所有镇外地点视野降低 |
