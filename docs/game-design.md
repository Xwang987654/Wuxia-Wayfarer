# 游戏设计总纲（GDD）

> 本文档是游戏设计的中心枢纽，各子系统的详细设计见 `systems/` 目录。

## 一、游戏概述

| 项目 | 描述 |
|------|------|
| 游戏名 | 武侠·自由人生（暂定） |
| 类型 | 武侠模拟经营 + 养成 + 轻度探索 |
| 视角 | 2.5D 斜俯视（45度等距） |
| 画风 | 像素风 |
| 平台 | 待定（优先PC/Web） |
| 参考 | 烟雨江湖、星露谷物语、太吾绘卷 |

## 二、核心循环

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│  经营赚钱    │────▶│  修炼成长    │────▶│  探索事件    │
│  交易/开店   │     │  学武功     │     │  随机遭遇    │
│  制造/采药   │     │  提属性     │     │  江湖传闻    │
└─────────────┘     └─────────────┘     └─────────────┘
       │                                       │
       │            ┌─────────────┐            │
       └───────────▶│   NPC社交    │◀───────────┘
                    │  交易/对话   │
                    │  关系/好感   │
                    └─────────────┘
```

玩家每天在这些活动中自由分配时间，没有强制任务。

## 三、人生路线

玩家可自由选择路线，路线之间**不互斥**。

### 3.1 江湖散人
- 自由探索地图
- 接悬赏任务
- 行商跑买卖
- 声望：侠义 ↔ 恶名

### 3.2 门派弟子
- 拜入武馆/门派
- 学习专属武功
- 门派任务、比武
- 成长：弟子 → 长老 → 掌门

### 3.3 衙门捕快
- 入职衙门
- 查案、断案
- 维护治安
- 官府俸禄

### 3.4 经商
- 开铺子（药铺/铁匠/茶馆）
- 跑商（城镇间价差）
- 炼丹/锻造出售
- 目标：富甲一方

> 详细设计见 `systems/` 各系统文档。

## 四、系统概览

### 4.1 时间系统
- 1天 = 10分钟（现实时间）
- 日夜5时段（清晨/上午/午后/傍晚/夜晚）
- 四季循环，每6天切换
- 随机天气影响游戏玩法

> 详见 `systems/time-system.md`

### 4.2 战斗系统
- 回合 + 自动混合
- 战前策略为主（装备/武功/丹药搭配）
- MVP阶段：文字结果输出，无战斗场景
- 后续扩展：正式回合制战斗界面

> 详见 `systems/combat-system.md`

### 4.3 经济系统
- 集市买卖（价格每日浮动）
- NPC差异化收购价
- 跑商（跨城镇贸易，有劫匪风险）
- 开铺子（进货→定价→自动售卖）
- 轻量制造（采药→炼丹，挖矿→打铁）

> 详见 `systems/economy-system.md`

### 4.4 NPC系统
- 分类：功能NPC、师傅NPC、任务NPC、探索NPC、敌对NPC
- 对话：选择式，影响好感和声望
- 交易：买卖物品，价格受供需和好感影响
- 关系：好感度系统
- 日程：NPC有独立作息

> 详见 `systems/npc-system.md`

### 4.5 任务系统
- 悬赏板（自取任务）
- NPC支线任务（好感解锁）
- 官府任务（衙门路线专属）
- 门派任务（门派路线专属）
- 随机事件触发

> 详见 `systems/quest-system.md`

### 4.6 声望系统
- 侠义 ↔ 恶名（双向量表）
- 门派声望（各门派独立）
- 官府声望
- 商誉（经商路线）

> 详见 `systems/reputation-system.md`

## 五、角色属性

### 5.1 基础属性

| 属性 | 说明 |
|------|------|
| 生命 | 归零则昏迷，回到客栈恢复 |
| 体力 | 行动消耗，休息恢复 |
| 内力 | 释放武功消耗，打坐恢复 |

### 5.2 战斗属性

| 属性 | 说明 |
|------|------|
| 攻击 | 物理伤害基础值 |
| 防御 | 减免物理伤害 |
| 轻功 | 影响闪避、逃跑、先手 |

### 5.3 社交属性

| 属性 | 说明 |
|------|------|
| 口才 | 对话选项、交易议价 |
| 悟性 | 修炼速度、学习武功效率 |
| 根骨 | 基础成长潜力 |

### 5.4 经营属性

| 属性 | 说明 |
|------|------|
| 眼力 | 发现低价好货、鉴别物品品质 |
| 手艺 | 制造品质、炼丹成功率 |
