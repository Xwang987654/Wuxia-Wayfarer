# 游戏内容生产规则

## 生产原则

1. **每个 NPC 都必须有玩法功能**：商店、任务发布、训练、情报、采集指引等，不能只站着
2. **每个任务必须有**：触发条件、目标、奖励、后果
3. **每个事件必须有**：选择和结果，选项不能是摆设
4. **每个地点必须承载**：NPC、事件、商店、采集点、任务入口中的至少一种
5. **每段对话应承担至少一种功能**：推进任务、提供线索、表现关系、打开商店、传递传闻、展示性格、改变状态
6. **不写无法落地的空泛设定**
7. **不随意扩大世界规模**：默认围绕清风镇、铁掌门、衙门、青竹帮、商旅、药铺、客栈、后山、黑市展开
8. **所有内容都指向玩法**

## 冲突处理规则

1. 如果用户要求与既有世界观冲突，必须指出冲突点
2. 不得为了满足用户要求而覆盖已设定世界观
3. 必须给出一个兼容世界观的改写方案
4. 如果必须新增设定，应标记为 `proposed`
5. 不得擅自新增大势力、新国家、隐藏超级组织、绝世高手
6. 如果 lore 文件缺失或无法读取，必须说明缺失内容，不得假装已读取

## 世界状态与 flags

flags 用于记录玩家选择、任务进度、NPC 状态、世界变化。任务、事件、对话必须通过 flags 控制前置条件和后续影响。

### 命名规范

- 小写英文 + 下划线
- 格式：`{类别}_{具体内容}_{状态}`

### 示例

```
quest_missing_medicine_started
quest_missing_medicine_completed
npc_doctor_trust_high
qingzhu_gang_warned_player
black_market_unlocked
player_joined_iron_palm
iron_palm_reputation_respected
```

### 使用要求

- 不要只写"玩家知道了真相"，要转化为 flag：`player_learned_herb_theft_truth`
- 任务前置条件用 flags 表达
- 对话分支用 flags 控制
- 事件触发用 flags 做条件

## 内容规模控制

1. 普通 NPC 背景不超过 200 字
2. 普通对话不超过 5 个节点
3. 普通任务不超过 3 个主要目标
4. 普通随机事件不超过 3 个选项
5. 单个地点描述不超过 80 字
6. 不要每个任务都写反转
7. 不要每个 NPC 都有惊天秘密
8. 清风镇阶段禁止写：飞天遁地、御剑飞行、起死回生、一掌碎山、千里传音、灭门惨案、绝世秘籍满地走

## 工程化输出规范

1. 默认输出 JSON，除非用户明确要求自然语言
2. JSON 外不要加大段解释
3. 文本字段可以中文
4. id 字段必须使用小写英文、数字、下划线
5. 所有引用必须使用 id，不得用中文名作为逻辑引用
6. 新增内容时必须写出关联地点、关联 NPC、前置条件、结果 flag
7. 状态变化统一使用 `effects` 结构，详见 `refs/data-contract.md` 中的 Effects 结构定义
8. 如果新增内容暂时没有对应系统实现，也要使用可扩展字段表达
9. 不要输出不合法 JSON
10. 设计意图放到 `design_note` 字段里

## Authoring 与 Runtime 分离

模板中包含两类字段：
- **runtime**：Godot 运行时加载的字段（id、name、type、price 等）
- **authoring**：策划辅助字段，不进入运行时（design_note、background、secret、full_story 等）

两类字段在 `refs/data-contract.md` 中有明确分类。生成时两类都保留，但 authoring 字段在导入 Godot 时可被忽略。

## 货币与时间

### 货币

- 底层统一使用 `copper`（文）
- 1 两 = 1000 文
- 文本展示可写"文"或"两"
- 参考物价：
  - 一碗面：5-10 文
  - 一把普通铁剑：500-800 文
  - 小任务报酬：20-300 文
  - 普通悬赏：1-5 两
  - 大任务：10 两以上

### 时间

一天分为五个时段，NPC 日程使用以下名称，不使用数字编号：

| 时段 | 英文 key | 大致对应 |
|------|----------|----------|
| 清晨 | `morning` | 卯时 |
| 正午 | `noon` | 午时 |
| 午后 | `afternoon` | 未时-申时 |
| 黄昏 | `evening` | 酉时-戌时 |
| 夜晚 | `night` | 亥时-寅时 |
