# 声望系统

## 概述

声望代表玩家在江湖中的名声和地位。声望影响NPC态度、可触发的事件和可解锁的内容。

## 声望类型

### 侠义声望（双向量表）

```
恶名 ◄──────────────┼──────────────► 侠义
-100                 0                +100

恶名区：-100 ~ -1   │  侠义区：1 ~ 100
NPC恐惧/警惕        │  NPC尊敬/信任
官府追捕            │  官府嘉奖
山贼友好            │  山贼回避
```

**增加侠义**：
- 完成正义任务（护送、清剿山贼）
- 救助NPC
- 拒绝贿赂

**增加恶名**：
- 抢劫NPC
- 杀害无辜
- 加入山贼
- 接受脏活

### 门派声望（各门派独立）

每个门派有独立的声望值：

| 门派 | 声望效果 |
|------|---------|
| 铁掌门 | 影响铁掌门武功学习、门派任务 |
| 后续门派 | 各自独立 |

**提升方式**：完成门派任务、比武获胜、为门派争光

### 官府声望

| 区间 | 效果 |
|------|------|
| 低 | 普通百姓 |
| 中 | 获得官府信任、接高级任务 |
| 高 | 晋升捕头、获得官府资源 |

**提升方式**：完成官府任务、破案、维护治安

### 商誉（经商路线）

| 区间 | 效果 |
|------|------|
| 低 | 普通价格交易 |
| 中 | NPC给优惠价、大客户折扣 |
| 高 | 稀有供货渠道、批发价格 |

**提升方式**：大量交易、诚信经营、帮助商人NPC

## 声望影响

### 对NPC的影响

```
侠义高 → 正派NPC友好，反派NPC敌对
恶名高 → 正派NPC警惕，反派NPC友好
官府声望高 → 衙门NPC尊敬，可接高级任务
商誉高 → 商人NPC给优惠
```

### 对事件的影响

| 声望 | 触发事件 |
|------|---------|
| 侠义≥50 | 被邀请参加英雄大会 |
| 侠义≥80 | 被推举为盟主候选人 |
| 恶名≥50 | 山贼邀请入伙 |
| 恶名≥80 | 官府发布通缉令 |
| 官府声望≥50 | 参与重大案件 |
| 商誉≥50 | 大商会邀请合作 |

### 对路线的影响

| 声望 | 效果 |
|------|------|
| 恶名高 | 不能加入衙门（需先洗白） |
| 侠义高 | 拜师更容易（门派欢迎） |
| 官府声望高 | 衙门路线晋升快 |
| 某门派声望低 | 不能加入该门派 |
