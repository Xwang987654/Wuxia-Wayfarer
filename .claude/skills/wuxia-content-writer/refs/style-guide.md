# 风格指南

## 禁止的 AI 套话

以下写法已被用烂，避免使用：

1. "命运的齿轮开始转动"及一切宿命论套话
2. "眼中闪过一丝XX"——模板化眼神描写
3. "嘴角微微上扬"——已被用烂
4. "不禁""竟然""居然"连续使用——虚词堆砌
5. "这一刻，他明白了..."——AI 最爱句式
6. "仿佛整个世界都..."——夸张环境渲染
7. "深邃的目光""坚毅的面庞""挺拔的身姿"——模板化外貌描写
8. 连续堆砌三个以上四字成语

**注意**：不禁止所有成语、眼神描写或"似乎""仿佛"等正常词汇。禁止的是模板化、堆砌式使用。自然出现在对话或叙述中没问题。

## 对话原则

**对话要像真人说话**：
- 铁匠不会说"鄙人从事锻造行业多年"，他会说"打了三十年铁了"
- 捕快不会说"维护治安是吾等职责"，他会说"这镇上的事，归我管"
- 老百姓不会说"近日物价飞涨令人忧心"，他会说"盐又涨了，这日子没法过了"

**不同身份的说话方式**：

| 身份 | 特点 | 示例 |
|------|------|------|
| 铁匠/匠人 | 话少，直来直去，用数字 | "这把剑，二十两。""不行，最少十八。" |
| 药铺/郎中 | 谨慎，用术语但会解释 | "这味药叫半夏，有毒，但炮制过后能化痰。" |
| 客栈掌柜 | 圆滑，见人说人话 | "客官看着面生，第一次来清风镇吧？楼上请。" |
| 门派掌门 | 有威严不装腔 | "拳法的事，练了才知道。光嘴上说没用。" |
| 衙门捕快 | 公事公办，偶尔透不满 | "这事我记下了。你先回去，有消息叫你。" |
| 江湖散人 | 带江湖气但不过分 | "在下路过此地，讨碗水喝。" |
| 乞丐/隐士 | 随意，偶尔说深话 | "老头子我啊，年轻时候也想过闯荡江湖。后来嘛，膝盖不好，就算了。" |
| 商人 | 三句不离本行 | "这批货从临河镇运过来，光路上就走了三天。" |
| 村民/普通人 | 朴实，关注日常 | "今年雨水好，田里收成不错。" |

## 叙述原则

- 简洁有力："天快黑了。集市上的摊贩开始收摊。"，不写"夕阳的余晖洒在古老的青石板路上"
- 情绪克制："他没说话，转身走了。"，不写"心中涌起难以言喻的悲痛"
- 只写玩家能看到、能互动的东西
- 不写意境写实物：不写"岁月的痕迹"，写"墙皮剥落了大半"
- 每个地点一两个标志性细节就够了
- 天气和时间要影响描写：雨天写泥泞，夜晚写灯火，冬天写呵气

## 人物逻辑

- 江湖人讲规矩、讲面子、讲辈分
- 普通人怕事、怕官、怕花钱
- 商人算账、算人情、算风险
- 每个人都有自己的日子要过，不是为了配合主角才存在的

## 命名规范

### 人名

- 清风镇附近的人名要朴素，不要过度仙侠化
- 普通 NPC：王铁匠、陈捕头、周掌柜、李二河、赵铁山
- 可用职业代称：张铁匠、李药师、刘屠夫
- 女性 NPC：王婶、春娘、柳姑娘（年轻未婚）、X氏（已婚）

### 武学名

- 普通武学短而实用：铁掌三式、伏虎拳、穿林步、青竹刀法
- 绝学可稍有气势但不滥用"玄、天、神、魔、帝、圣"
- 好名字例：破碑手、缠丝劲、回风剑、八步赶蝉

### 地名

- 要像真实地方：清风镇、临河镇、落霞渡、青石坡
- 不要全是"缥缈峰、无极谷、玄天岭"

### 任务名

- 像游戏任务名，不太网文
- 好例：失踪的药材、后山贼患、铁掌门的考验
- 差例：命运的交织！惊天阴谋浮出水面

### 物品名

- 朴素有生活感：铁剑、止血散、狗皮膏药、粗布护腕
- 好一点的东西：百锻钢刀、金疮药、熟铜护心镜
