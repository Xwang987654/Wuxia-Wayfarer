# 项目愿景

「武侠·自由人生」的长期方向是低武侠、养生向的小人物生活模拟。玩家不是天命主角，而是来到清风镇的普通人，在镇上认识人、做事、攒钱、听消息，慢慢决定自己的日子怎么过。

当前阶段只服务第一版 demo：证明清风镇的最小生活循环能玩、能读、能被 Godot 从 JSON 数据驱动起来。

## 当前目标

- 用清风镇作为唯一舞台。
- 让玩家完成一次完整的“行动 -> 对话 -> 接任务 -> 收集 -> 交付 -> 奖励 -> 时间推进”循环。
- 保持低武侠质感：有人情、账目、规矩和烟火气，不展开奇观和大阴谋。
- 建立稳定的 `data/` 文档与运行数据结构，避免多套口径并行。

## 长期基调

- 世界是大燕朝承平年间，表面安稳，地方有小摩擦。
- 清风镇位于江南与中原交界，是商旅经过的小镇。
- 江湖存在，但第一版 demo 只让玩家看到边缘痕迹。
- 武侠味来自人物说话、镇上规矩、地方传闻和生活选择，而不是高强度战斗。
