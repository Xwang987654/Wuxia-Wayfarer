# 天下背景

## 朝代

大燕朝，立国已逾二百年。

开国时以武定天下，太祖本人就是马上皇帝，麾下猛将如云。立国后偃武修文，武将卸甲，文官治国。二百年下来，朝廷的刀锋钝了，但天下富了。

如今是大燕朝承平年间。没有大规模战乱，百姓日子过得去。但"过得去"不等于"过得好"——赋税年年涨，地方官越来越跋扈，民间怨气在积攒。

### 中原
天下正中，自古帝王之都所在。几大门派扎根于此，武林正统在此传承。中原人自视甚高，视其他地方为"偏僻之地"。

### 江南
水乡泽国，物产丰饶，商贸繁荣。江南富甲天下，但武学传承不如中原深厚。江南人重利轻义，善于经商，却也出了不少以柔克刚的武学流派。

清风镇位于江南与中原的交界地带，扼守南北要道，是商旅必经之地。

### 塞外
苦寒之地，游牧部落逐水草而居。塞外人性情彪悍，武学走刚猛路线。边疆时有摩擦，但大规模冲突已多年未有。

### 南疆
瘴气弥漫的热带丛林，部族众多，风俗奇异。南疆武学多奇招怪式，中原武林视之为旁门左道。但南疆高手从不在意中原人的看法。

### 关中
古都所在，大燕朝京师长安在此。朝廷中枢、禁军、官府武学总署皆在此地。

## 江湖与朝廷

二百年来，朝廷对江湖的态度经历了几个阶段：

```
开国初期：利用
  → 太祖靠江湖势力打天下，封赏功臣
  → 江湖人有官做，朝廷和江湖是一家人

中期：疏远
  → 文官上位，武将靠边
  → 江湖人被排挤出朝堂
  → 各大门派自立门户，不再与朝廷来往

如今：忌惮
  → 江南商贸繁荣，门派势力壮大
  → 朝廷担心门派勾结地方势力
  → 设立"武学总署"，名义上管理天下武学
  → 实际上想控制江湖门派

  大门派：根基深厚，朝廷动不了
  小门派：实力有限，容易被渗透
  散人：无组织，朝廷最看不上的，也最自由
```

## 普通人的生活

不是所有人都闯江湖。大多数人是普通百姓：

- **农人**：种地交赋税，日子平淡
- **商人**：江南最多，走南闯北，有钱但社会地位不高
- **匠人**：铁匠、木匠、石匠，靠手艺吃饭
- **读书人**：科举是正途，但录取率极低
- **江湖人**：学过武的统称，但大多数人学武只是为了看家护院

对普通人来说，"江湖"是茶余饭后的谈资，是说书人口中的传奇，是遥远的、与自己无关的世界。

直到有一天，江湖找上了门。
