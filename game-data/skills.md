# 武功 / 技能

## 概述

武功是战斗的核心，也是角色成长的重要体现。武功通过拜师学艺、任务奖励、探索发现等方式获取。

## 武功分类

### 按武器类型

| 类型 | 说明 | 代表武功 |
|------|------|---------|
| 拳法 | 空手格斗 | 铁砂掌、伏虎拳 |
| 掌法 | 掌力攻击 | 降龙掌（远期） |
| 剑法 | 用剑攻击 | 基础剑法 |
| 刀法 | 用刀攻击 | 基础刀法 |
| 暗器 | 远程攻击 | 飞刀术 |
| 轻功 | 闪避/逃跑 | 基础轻功 |
| 内功 | 增益/恢复 | 基础吐纳 |

### 按效果类型

| 类型 | 说明 |
|------|------|
| 攻击 | 造成伤害 |
| 防御 | 减免伤害 |
| 增益 | 临时提升属性 |
| 恢复 | 恢复生命/内力 |
| 控制 | 降低敌人属性 |
| 逃跑 | 提高逃跑成功率 |

## 武功属性

| 属性 | 说明 |
|------|------|
| 名称 | 武功名 |
| 类型 | 拳/剑/刀/... |
| 伤害倍率 | 基础伤害乘数 |
| 内力消耗 | 使用消耗内力 |
| 暴击率 | 触发暴击的概率 |
| 特殊效果 | 如：降低敌人防御、恢复自身生命 |
| 学习条件 | 需要的属性/前置武功 |
| 来源 | 从谁那里学/在哪里找到 |

## 武功装备

- 玩家可同时装备 **3个主动武功** + **1个内功**
- 主动武功：战斗中可选择使用
- 内功：被动效果，持续生效
- 可随时在菜单中更换装备的武功

## 武功升级

| 等级 | 效果提升 | 升级方式 |
|------|---------|---------|
| 初学 | 基础效果 | 学习获得 |
| 小成 | 伤害+20% | 使用次数达标 |
| 大成 | 伤害+50%，解锁特殊效果 | 使用次数达标+悟性要求 |
| 精通 | 伤害+100%，特殊效果增强 | 极高使用次数+特殊条件 |

## 战斗中的武功使用

```
战斗前准备：
┌──────────────────────────────────────┐
│  装备武功                            │
│                                      │
│  主动武功栏：                         │
│  ① 铁砂掌  ② 基础剑法  ③ 空          │
│                                      │
│  内功栏：                             │
│  基础吐纳                             │
│                                      │
│  已学武功：                           │
│  铁砂掌、伏虎拳、铁布衫、基础剑法...  │
│                                      │
│  [确认]                              │
└──────────────────────────────────────┘
```

战斗中（自动）：
- 系统按优先级选择使用武功
- 优先使用高伤害武功（内力足够时）
- 内力不足时使用基础武功
- 可设置战术偏好（激进/保守/平衡）
