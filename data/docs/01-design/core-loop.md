# Core Loop

第一版 demo 的核心循环必须短、清楚、可测试。

## 最小循环

1. 玩家在镇口醒来或进入清风镇。
2. 玩家移动到青石街，看到可去地点。
3. 玩家与药师林对话，接到采草药任务。
4. 玩家去溪边草坡采集 3 份草药。
5. 时间推进一个时段。
6. 玩家返回林家药铺交付草药。
7. 玩家获得铜钱或馒头奖励。
8. 玩家可以在背包或状态中看到变化。

## 交互原则

- 每次行动只产生一个清楚结果。
- demo 阶段不做失败分支，避免系统提前膨胀。
- 对话选项保持少量，最多 3 个。
- 任务只需要一个目标和一个完成条件。
- 时间推进只需要让玩家意识到“一天在流动”。

## 体验目标

玩家应感到自己在一个有人味的小镇里做了一件小事，而不是被系统推着完成教程。
