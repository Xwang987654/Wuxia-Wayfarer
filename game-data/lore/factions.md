## 朝廷与官府

### 武学总署

朝廷设立的官方武学管理机构，名义上"规范天下武学"，实际上想控制江湖势力。

```
┌──────────────────────────────────────────────────────────────┐
│  武学总署                                                     │
│                                                              │
│  总部：长安                                                   │
│  首领：武学总署令（朝廷命官）                                 │
│                                                              │
│  职能：                                                       │
│  • 管理各地衙门的武学教育                                     │
│  • "扶持有潜力的小门派"（实际是渗透）                        │
│  • 收集江湖情报                                               │
│  • 打压不听话的江湖势力                                       │
│                                                              │
│  手段：                                                       │
│  • 给小门派送钱送资源（拉拢）                                 │
│  • 安排"教头"进入小门派（渗透）                              │
│  • 对大门派示好但无法真正控制                                 │
│  • 利用官府武者处理"江湖纠纷"（彰显存在感）                  │
│                                                              │
│  现状：                                                       │
│  • 大门派看穿了但不戳破                                       │
│  • 部分小门派已经被"扶持"，实为傀儡                          │
│  • 部分小门派在抵抗                                           │
│  • 散人不在他们视野内（太零散，不值得管）                     │
│                                                              │
└──────────────────────────────────────────────────────────────┘
```

## 大门派（MVP 不出现，仅世界设定）

天下公认的大门派有二到三个，皆在中原：

```
┌──────────────────────────────────────────────────────────────┐
│  天元宗（中原）                                               │
│  ────────────                                                │
│  天下第一门派，传承三百年                                     │
│  内功心法冠绝天下，弟子遍天下                                 │
│  从不参与江湖纷争，超然物外                                   │
│  朝廷不敢动，也动不了                                         │
│                                                              │
│  特点：正统、底蕴深、弟子要求高                               │
│                                                              │
├──────────────────────────────────────────────────────────────┤
│  沧澜剑派（中原/齐鲁交界）                                    │
│  ────────────────                                            │
│  以剑法闻名，门下弟子多为剑客                                 │
│  风格飘逸，追求"剑道"而非"武力"                              │
│  与天元宗分庭抗礼                                             │
│                                                              │
│  特点：潇洒、重剑道、不拘小节                                 │
│                                                              │
├──────────────────────────────────────────────────────────────┤
│  镇远镖局（关中/边塞）                                        │
│  ────────────────                                            │
│  严格说不算"门派"，但势力庞大                                 │
│  走镖、押运、边贸护卫                                         │
│  与朝廷关系最近，但保持独立                                   │
│                                                              │
│  特点：务实、重义气、商业头脑                                 │
│                                                              │
└──────────────────────────────────────────────────────────────┘

这三大势力对清风镇的影响：
  • 偶有弟子路过清风镇（带来远方消息）
  • 铁掌门的武功可能与某大门派有渊源
  • 官府渗透小门派，背后是想制衡大门派
```

## 小门派（百花齐放）

小门派遍布天下，各有特色。它们的特点是：

```
• 规模小：少则十几人，多则百人
• 地域性强：多在地方上经营，不跨区域
• 特色鲜明：各有独门武功或理念
• 生存压力大：需要地方支持，容易被官府拉拢
```

### 清风镇及周边的小门派

#### 铁掌门

```
┌──────────────────────────────────────────────────────────────┐
│  铁掌门                                                       │
│  位置：清风镇                                                 │
│  规模：约30人                                                 │
│  掌门：赵铁山（60岁）                                         │
│                                                              │
│  历史：                                                       │
│  • 创建于六十年前，创始人赵铁山之父                           │
│  • 起初只是一个武馆，教镇上年轻人练拳                         │
│  • 赵铁山接手后发展壮大，有了门派雏形                         │
│  • 靠"铁砂掌"和"伏虎拳"在清风镇立足                         │
│                                                              │
│  特色：                                                       │
│  • 刚猛路线，以力取胜                                         │
│  • 内功走阳刚路线，真气浑厚                                   │
│  • 弟子多为普通人家孩子，学费不高                             │
│  • 门规不严，相对自由                                         │
│                                                              │
│  现状：                                                       │
│  • 清风镇最大的武力组织                                       │
│  • 与衙门关系：表面客气，暗中较劲                             │
│  • 官府曾想"扶持"铁掌门，被赵铁山婉拒                       │
│  • 赵铁山在观望局势，不站队                                   │
│                                                              │
│  问题：                                                       │
│  • 后继乏人——没有特别出色的弟子                              │
│  • 赵铁山年事已高，谁来接班是个问题                           │
│  • 门派收入有限，全靠学费                                     │
│                                                              │
└──────────────────────────────────────────────────────────────┘
```

## 其他势力

### 商会

```
江南商贸繁荣，商人自发组成商会，互通有无。
商会不涉及武力，但有钱。有钱就有影响力。
清风镇有一个小商会分号，由刘商人张罗。
```

### 乞丐/隐士

```
江湖中有一类人：看似乞丐，实为隐世高手。
或避世隐居，或游戏人间。
清风镇的老乞丐可能就是这类人。
```
