# 时间系统

## 概述

时间是游戏世界运转的基础。四季更替、日夜交替、NPC作息、经济波动都依赖时间系统驱动。

## 时间规格

| 参数 | 值 |
|------|-----|
| 1天（游戏内） | 10分钟（现实） |
| 1季 | 6天 = 1小时（现实） |
| 1年（四季循环） | 24天 = 4小时（现实） |

## 日夜时段

一天分为5个时段，每段2分钟（现实）：

| 时段 | 时间范围 | 视觉表现 | 游戏影响 |
|------|---------|---------|---------|
| 清晨 | 0:00 - 2:00 | 天色微亮，雾气 | 修炼加成（吐纳最佳时机） |
| 上午 | 2:00 - 4:00 | 阳光明媚 | 商铺开门，可交易 |
| 午后 | 4:00 - 6:00 | 日光正盛 | NPC活跃，可接任务 |
| 傍晚 | 6:00 - 8:00 | 夕阳西下 | 集市热闹，稀有物品出现 |
| 夜晚 | 8:00 - 10:00 | 月色/灯光 | 衙门关门，黑市开张，夜间事件 |

## 四季

### 基础特性

| 季节 | 视觉变化 | 全局影响 |
|------|---------|---------|
| 春 | 嫩绿、花开 | 草药生长旺盛，门派招弟子 |
| 夏 | 翠绿、阳光强烈 | 烈日体力消耗加快，商路繁荣 |
| 秋 | 金黄、落叶 | 丰收季产量翻倍 |
| 冬 | 灰白、积雪 | 大雪封山部分区域不可去 |

### 季节专属事件

| 季节 | 事件 |
|------|------|
| 春 | 门派招收弟子、春雨修炼加成 |
| 夏 | 江湖比武大会、跑商利润高 |
| 秋 | 门派论剑、秋猎活动 |
| 冬 | 内功修炼最佳时机、年末事件（还债/分红/门派考核） |

## 天气系统

每个季节有若干天气类型，随机触发。天气持续时间随机（半天到两天）。

### 天气类型

| 天气 | 季节 | 概率 | 影响 |
|------|------|------|------|
| 晴天 | 全季 | 高 | 无特殊影响 |
| 细雨 | 春 | 中 | 药铺生意好（风湿药需求↑） |
| 春雷 | 春 | 低 | 触发特殊剧情事件 |
| 雾气 | 春/秋 | 中 | 探索视野降低 |
| 暴雨 | 夏 | 中 | 集市客流少，某些NPC不出门 |
| 酷暑 | 夏 | 中 | 体力消耗加快，茶馆火爆 |
| 雷阵雨 | 夏 | 低 | 短暂影响出行 |
| 秋风 | 秋 | 中 | 轻功修炼加成 |
| 薄雾 | 秋 | 中 | 探索视野略降 |
| 大雪 | 冬 | 中 | 道路封闭，适合室内修炼 |
| 霜冻 | 冬 | 低 | 药材保存时间延长 |
| 寒风 | 冬 | 中 | 体力消耗加快 |

### 天气对系统的影响

```
天气 ──┬──▶ 经济：影响商品供需和价格
       ├──▶ NPC：影响NPC出没和行为
       ├──▶ 探索：影响可到达区域
       ├──▶ 战斗：某些天气有战斗修正
       └──▶ 修炼：某些天气有修炼加成/减益
```

## 时间控制

- **正常速度**：默认10分钟/天
- **快进**：玩家可按键跳过当前时段（如只想练功不想看日出）
- **等待**：在特定地点可选择"等待到某时段"（如在客栈等到夜晚）
- **暂停**：菜单打开时时间暂停

## 时间触发机制

### 定时触发
- 每天清晨：商铺刷新库存、NPC重置日程
- 每季第一天：季节切换、季节事件检查
- 每年冬季末：年度事件

### 条件触发
- 特定天气 + 特定地点 = 触发事件
- 特定时段 + 特定NPC位置 = 触发对话
- 季节 + 声望条件 = 触发剧情
