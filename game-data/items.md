# 物品 / 道具

## 物品分类总览

| 类别 | 说明 | 示例 |
|------|------|------|
| 武器 | 攻击用装备 | 剑、刀、拳套、暗器 |
| 防具 | 防御用装备 | 衣服、护腕、鞋子 |
| 丹药 | 消耗品，恢复/增益 | 回血丹、回气丹 |
| 材料 | 制造原料 | 草药、矿石、布料 |
| 食物 | 恢复体力 | 包子、酒、茶 |
| 书籍 | 提升属性/学习武功 | 武功秘籍、诗集 |
| 任务物品 | 任务专用 | 信件、令牌、赃物 |
| 特殊物品 | 稀有/彩蛋物品 | 古董、藏宝图 |

## 物品属性

每个物品包含以下属性：

| 属性 | 说明 |
|------|------|
| 名称 | 物品名 |
| 描述 | 物品背景故事 |
| 类别 | 武器/防具/丹药/... |
| 品质 | 普通/精良/稀有/传说 |
| 基础价格 | 铜钱价格 |
| 重量 | 影响携带数量 |
| 效果 | 具体数值效果 |
