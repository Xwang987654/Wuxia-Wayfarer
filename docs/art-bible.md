# 美术圣经

## 整体风格

**像素风 · 2.5D 等距视角 · 中国古风武侠**

画面呈现45度斜俯视的等距视角，像素风格，色调温暖，带有中国江南水乡的柔美感。

## 画面规格

| 项目 | 规格 |
|------|------|
| 视角 | 2.5D 等距（45度斜俯视） |
| 像素密度 | 角色 32x32 ~ 64x64，Tile 32x32 |
| 色盘 | 每个角色/物品限制色数，保持像素感 |
| 分辨率 | 待定（取决于技术栈选择） |

## 场景设计

### 建筑风格

- **民居**：白墙灰瓦，木门木窗，江南水乡风格
- **商铺**：招牌旗幡，灯笼高挂，门前有货摊
- **武馆**：牌匾、石狮、练武场、木桩
- **衙门**：石狮、鸣冤鼓、官府牌匾、肃穆感
- **客栈**：二楼结构，门前酒旗，灯火温暖

### 地面与环境

- **镇内**：石板路、青砖路面、小桥流水
- **镇外**：泥土小径、竹林、溪流、山石
- **季节变化**：同场景四季不同色调（春绿/夏翠/秋黄/冬白）

### 天气视觉效果

| 天气 | 视觉表现 |
|------|---------|
| 晴天 | 明亮，有阳光方向感 |
| 细雨 | 半透明雨丝，地面反光 |
| 暴雨 | 密集雨线，画面略暗 |
| 大雪 | 白色飘落粒子，地面积雪 |
| 雾气 | 远景模糊，近景正常 |
| 夜晚 | 整体暗色调，灯光区域亮起 |

## 角色设计

### NPC 视角
- 3/4 俯视角（与场景一致的等距视角）
- 像素风格，清晰可辨识的身份特征

### 角色类型视觉区分

| 类型 | 视觉特征 |
|------|---------|
| 武馆师傅 | 传统武服，束发，体格健壮 |
| 门派弟子 | 统一制式服装，佩剑 |
| 衙门捕快 | 官服，腰牌，官帽 |
| 商人 | 绸缎衣裳，算盘/账本 |
| 铁匠 | 围裙，肌肉发达，手持锤子 |
| 药铺老板 | 长衫，药箱/药罐 |
| 客栈掌柜 | 围裙，毛巾搭肩 |
| 山贼/盗匪 | 粗布衣，面巾，凶悍表情 |
| 江湖散人 | 斗笠/竹笠，随意装束 |
| 村民/路人 | 普通布衣，各有差异 |

### 玩家角色
- 开局可选择基础外观
- 装备变化体现外观差异（武器、衣服）
- 后续可扩展捏脸系统

## UI 风格

- 像素风 UI，与场景风格统一
- 按钮/面板使用木质或纸质质感
- 字体：像素字体（中文需专门选择可读性好的像素字体）
- 图标：简洁明了，像素风格小图标

## AI 素材生成规范

### 统一 Prompt 模板

生成素材时使用以下基础 prompt，根据具体需求追加描述：

**场景 Prompt 基础模板：**
> 2.5D isometric pixel art, 45-degree top-down view, Chinese ancient martial arts town, warm color palette, clean pixel art style, 32x32 tile, retro RPG game aesthetic, no text, no UI overlay

**角色 Prompt 基础模板：**
> Pixel art character sprite, 64x64, 3/4 top-down isometric view, Chinese wuxia style, clean outlines, limited color palette, retro RPG game character, no background, transparent background

### 生成后处理
1. 统一色盘：将AI生成的素材色调调整为项目调色板
2. 尺寸统一：确保所有同类素材尺寸一致
3. 像素对齐：检查像素网格对齐，消除模糊
4. 去除瑕疵：AI生成的文字、畸形部分手动修正
