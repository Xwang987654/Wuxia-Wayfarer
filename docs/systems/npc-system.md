# NPC 系统

## 概述

NPC 是江湖世界的灵魂。每个NPC都有自己的身份、作息、喜好和关系网，而非单纯的功能性存在。

## NPC 分类

| 类型 | 职责 | 示例 |
|------|------|------|
| 功能NPC | 提供基础服务（交易/治疗/住宿） | 铁匠、药铺老板、客栈掌柜 |
| 师傅NPC | 拜师学艺、门派/组织任务 | 武馆掌门、衙门捕头 |
| 任务NPC | 提供任务、剧情推动 | 商人、村民、官差 |
| 探索NPC | 随机出现，提供事件/彩蛋 | 路人、乞丐、神秘人 |
| 敌对NPC | 战斗遭遇 | 山贼、盗匪 |

## NPC 属性

| 属性 | 说明 |
|------|------|
| 身份 | NPC的社会角色和背景 |
| 位置 | 常驻地点 |
| 日程 | 不同时段的行为 |
| 好感 | 对玩家的态度（-100 ~ 100） |
| 库存 | 可交易的物品（商人NPC） |
| 对话树 | 可触发的对话内容 |
| 关系网 | 与其他NPC的关系 |

## NPC 日程

每个NPC有自己的日常作息，按时段变化：

### 示例：铁匠张大锤

| 时段 | 位置 | 行为 |

## 好感系统

### 好感度范围

| 区间 | 态度 | 效果 |
|------|------|------|
| -100 ~ -50 | 敌对 | 拒绝交易、可能攻击 |
| -50 ~ -20 | 冷淡 | 交易价格差、对话敷衍 |
| -20 ~ 20 | 中立 | 正常交互 |
| 20 ~ 50 | 友好 | 交易优惠、透露信息 |
| 50 ~ 100 | 亲密 | 特殊帮助、赠送物品、专属任务 |

### 好感度变化

| 行为 | 好感变化 |
|------|---------|
| 完成NPC任务 | +5 ~ +20 |
| 赠送NPC喜欢的物品 | +3 ~ +10 |
| 在NPC处消费 | +1 ~ +3 |
| 对NPC无礼/威胁 | -5 ~ -20 |
| 帮NPC的敌人 | -10 ~ -30 |
| 伤害NPC | -50 |

### 好感度解锁

| 好感等级 | 解锁内容 |
|---------|---------|
| 中立 | 基础交易和对话 |
| 友好 | NPC支线任务、透露江湖传闻 |
| 亲密 | 专属武功/物品传授、特殊帮助 |

## NPC 交易

### 交易规则
- 每个NPC有基础库存，定期刷新
- 价格受好感度影响（好感越高越便宜）
- 某些稀有物品只在好感达到一定等级后才出售
- NPC有自己的收购偏好（铁匠偏好矿石，药师偏好草药）

## NPC 对话系统

### 对话结构

```
NPC对话节点
   │
   ├── 基础对话（所有玩家可触发）
   │     "你好，需要什么？"
   │
   ├── 条件对话（需要满足条件）
   │     好感≥50："看你是个可造之材..."
   │     声望≥侠义："大侠，久仰久仰！"
   │     任务中："事情办得怎么样了？"
   │
   └── 选择分支（玩家做选择）
         ① "我想买把剑" → 交易
         ② "最近有什么消息？" → 江湖传闻
         ③ "告辞" → 结束
```

### 对话影响
- 某些对话选项影响好感度
- 某些对话选项影响声望
- 某些对话选项触发任务
- 某些对话选项改变NPC态度
